# No-Go Rules

Do not add any of the following unless the user explicitly asks:
- React dashboard
- n8n
- Prisma
- Viber channel support
- Facebook Messenger channel support
- Redis
- Queue-based architecture changes
- WebSocket realtime features
- destructive migrations
- table renames
- production credential changes
- secret printing
- `.env` commits

Do not change locked behavior during unrelated work.
Do not broaden a task beyond the requested scope.
